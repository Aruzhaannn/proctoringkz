# 🎓 ProctorAI — Зияткерлік Емтихан Прокторинг Платформасы (Қазақ тілі)

Нақты уақыттағы камерамен толық жұмыс жасайтын ЖИ негізіндегі прокторинг жүйесінің интерактивті фронтенд-прототипі.

## 📂 Жоба құрылымы

```
proktorai-kz/
├── index.html                      ← Кіру беті (бастапқы нүкте)
├── css/
│   ├── global.css                  ← Дизайн жүйесі (оригиналдан)
│   └── login.css                   ← Кіру беті стильдері
├── js/
│   └── login.js                    ← Кіру интерактивтілігі
└── pages/
    ├── student-dashboard.html      ← Студент кабинеті
    ├── exam-room.html              ← 🎥 НАҚТЫ КАМЕРАМЕН емтихан бөлмесі
    ├── teacher-dashboard.html      ← Оқытушы тікелей бақылауы
    └── admin-dashboard.html        ← Әкімші жүйе панелі
```

## 🚀 Іске қосу

`index.html` файлын кез-келген заманауи браузерде ашыңыз.

⚠️ Камера жұмыс жасауы үшін HTTPS немесе localhost қажет:
- VS Code Live Server қолданыңыз
- Немесе: `python -m http.server 8080` → http://localhost:8080

## 🎥 НАҚТЫ КАМЕРА — Негізгі мүмкіндік

`exam-room.html` бетінде:

### Жұмыс жасайтын нәрселер:
| Мүмкіндік | Сипаттама |
|-----------|-----------|
| 📷 **Нақты камера** | `getUserMedia()` WebRTC API арқылы шынайы камера |
| 🔄 **Айна режимі** | Бейне горизонталь айналдырылады (selfie) |
| 🟢 **Бет анықтау рамкасы** | Камера ашылғанда жасыл рамка пайда болады |
| 📊 **Сканерлеу сызығы** | Анимациялы бет сканерлеу визуализациясы |
| 🎙️ **Микрофон деңгейі** | AudioContext арқылы нақты уақыттағы дыбыс визуализациясы |
| 🖥️ **Қойынды ауыстыру** | `visibilitychange` — нақты анықтайды |
| 📋 **Ctrl+C / Ctrl+V** | `copy` / `paste` оқиғалары — нақты анықтайды және бұғаттайды |
| ⏱️ **Таймер** | Кері санақ таймері |
| 🤖 **ЖИ алдау баллы** | Бұзушылықтарға байланысты динамикалық 0-100% балл |
| 📝 **Іс-шаралар журналы** | Нақты уақыттағы оқиғалар жазбасы |

### Браузер рұқсаттары:
Емтихан бөлмесін ашқанда браузер камера мен микрофонға рұқсат сұрайды.
- ✅ **Рұқсат берсе** → нақты камера ағыны іске қосылады
- ❌ **Рұқсат бермесе** → анимациялы placeholder + қате хабарламасы

## 🔐 Кіру жүйесі

Кіру бетінде рөлді таңдап, кіру батырмасын басыңыз:
- **Студент** → Студент кабинеті
- **Оқытушы** → Тікелей бақылау панелі  
- **Әкімші** → Жүйе әкімшілігі

## 🛠️ Технологиялық стек (Frontend прототипі)

- HTML5 / CSS3 / Vanilla JavaScript
- **WebRTC** `getUserMedia()` — нақты камера
- **Web Audio API** — микрофон визуализациясы
- Google Fonts: Syne + DM Sans
- Сыртқы тәуелділіктер жоқ

## 📋 Техникалық спецификация (Жоспарланған Backend)

- Java 17, Spring Boot 3, Spring Security (JWT/OAuth2)
- PostgreSQL, JPA/Hibernate, Redis
- Apache Kafka (хабарламалар брокері)
- Python, OpenCV, TensorFlow/PyTorch (ЖИ модулі)
- Docker, Kubernetes, GitHub Actions (CI/CD)
